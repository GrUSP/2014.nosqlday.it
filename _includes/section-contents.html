What's this about?

NoSQL Day is a one day conference about all last generation databases. After so many years relationa databases are not the only solution anymore!

This conference started in 2011, in italian, organized by WEBdeBS in Brescia as a way to help developers to get in touch for the first time with new interesting solutions. In 2013, in italian, organized by GrUSP in Udine as a continuation to see how things were evolving.

Starting from 2014 edition GrUSP and PHP User Group Friuli decided to go international. This is the answer to lots of requests from foreign speakers and foreign attendees willing to be part of this event. A single day to learn a lot of cool new stuff and do some great networking.

GrUSP is a no profit associaton. This event is made possible thanks to our sponsors and to the work offered by the organizers for free.

Qouting WEBdeBS, this is an event "from the community for the community".
